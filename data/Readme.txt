Please place the data in the following folder structure:

data <-- current folder
├── [data_name_1] <-- change to the actual name of data
│   ├── Distance
│   │   └── distances.txt
│   └── Images
│       ├── Front
│       │   ├── Front_xxx.jpg
│       │   ├── ...
│       └── Rear
│           ├── Rear_xxx.jpg
│           ├── ...
└── [data_name_2]
    ├── ...

Please ensure that you adhere strictly to the specified naming convention. For example, name the file "distances.txt" instead of "distance.txt".
